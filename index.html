The Worldview Debugger

What is this program?

The Worldview Debugger is an educational tool that models a set of worldview "programs" and their disciplines. Use the top-right tabs to switch between views. Header controls appear beneath the tabs for quick actions like toggling Plain English, opening the Disciplines Reference, or rebooting the session.

  • Scan: Run a quick scan that detects general argument examples. Results appear in the Scan Console on the right — click an item to inspect and attempt a corrective patch.
  • Debug: Inspect a chosen worldview and run discipline-specific analysis. Apply proposed patches to mark disciplines as repaired.
  • Status: View repair status across all worldviews and a compact summary of quick-scan findings. Use the controls to refresh or reset quick-scan flags.

Tip: Use the "Plain English" toggle in the header to switch between technical diagnostics and a simpler description of proposed patches.
